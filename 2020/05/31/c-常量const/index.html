c++常量const

  1. 常量

    常量值不可更改

  2. 常量指针

    const type *

    不可通过常量指针修改其指向的内容

    type * const
    不可更改指针值

  3. 常量引用

    不能通过常引用去修改其引用的内容

  4. 常成员函数

    1. 类成员函数后面加const关键字
    2. 常成员函数不能修改对象的成员变量的值
      1. 除了静态成员变量，因为静态成员变量不属于某个对象
    3. 常成员函数不能调用类的非常成员函数
      1. 非常成员函数可能会修改成员变量的值
      2. 除了静态成员函数，因为静态成员函数不属于某个对象
